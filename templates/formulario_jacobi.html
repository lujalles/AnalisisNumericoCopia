Sobre el Método de Jacobi

El método de Jacobi es una técnica iterativa para resolver sistemas de ecuaciones lineales. Es especialmente útil para sistemas grandes y dispersos, donde los métodos directos como la eliminación gaussiana pueden ser computacionalmente costosos.

Para un sistema Ax = b, el método de Jacobi calcula cada componente de la solución usando:

\[ x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1, j \neq i}^{n} a_{ij} x_j^{(k)} \right) \]

El algoritmo funciona de la siguiente manera:

  1. Se elige un vector inicial x^{(0)} (generalmente un vector de ceros).
  2. Para cada iteración k, se calcula cada componente xi^{(k+1)} usando los valores de x^{(k)}.
  3. Se repite el proceso hasta que la diferencia entre iteraciones consecutivas sea menor que la tolerancia especificada o se alcance el número máximo de iteraciones.

Ejemplo

Consideremos el sistema:

\begin{align} 4x_1 - x_2 + x_3 &= 7 \\ -x_1 + 4x_2 - x_3 &= -1 \\ x_1 - x_2 + 4x_3 &= 15 \end{align}

Despejando cada variable:

\begin{align} x_1 &= \frac{1}{4}(7 + x_2 - x_3) \\ x_2 &= \frac{1}{4}(-1 + x_1 + x_3) \\ x_3 &= \frac{1}{4}(15 - x_1 + x_2) \end{align}

Partiendo de x^{(0)} = [0, 0, 0], las primeras iteraciones serían:

x^{(1)} = [1.75, -0.25, 3.75]

x^{(2)} = [1.875, 1.375, 4.25]

Y así sucesivamente hasta converger a la solución x = [2, 1, 4].

Ventajas:

  • Fácil de implementar y paralelizar
  • Bajo costo computacional por iteración
  • Adecuado para sistemas grandes y dispersos
  • No modifica la matriz original

Desventajas:

  • Convergencia lenta en comparación con otros métodos
  • No garantiza convergencia para cualquier sistema
  • Requiere que la matriz sea diagonalmente dominante para garantizar convergencia
  • Puede requerir muchas iteraciones para alcanzar la precisión deseada

Criterio de convergencia:

El método de Jacobi converge si la matriz A es diagonalmente dominante, es decir, si para cada fila i:

\[ |a_{ii}| > \sum_{j=1, j \neq i}^{n} |a_{ij}| \]

O si la matriz es definida positiva.
